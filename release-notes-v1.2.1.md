## Skill-MT v1.2.1

本次更新聚焦于 Codex 项目技能管理、导入导出体验与解析稳定性。

## Added
- 新增 **Codex Project Skills** 浏览与管理能力，可随项目目录一起加载。
- 新增创建技能时的**位置偏好记忆**，支持在个人/项目位置间快速切换。

## Improved
- 优化导入技能流程：支持项目级目标位置、更多本地化文案与更稳定的位置回退逻辑。
- 优化导出流程：改为直接导出 `.zip` 文件，路径与结果更直观。
- 优化列表与空状态展示，提升中英文界面一致性。

## Fixed
- 修复 `frontmatter` 字段在 YAML 隐式类型下可能被错误解析的问题。
- 修复 Codex 项目技能在部分 CRUD 场景下的路径安全判断问题。
- 修复创建技能时输入法未提交导致描述读取不完整的问题。
