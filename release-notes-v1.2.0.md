## Skill-MT v1.2.0

This release significantly expands provider support and improves settings usability and update workflow.

### Highlights
- Added **Codex Skill management** support alongside Claude.
- Added **settings categorization** with section switching for cleaner navigation.
- Added **custom root directory configuration** for both Claude and Codex skills.
- Improved **Chinese/English localization** coverage when switching languages.
- Added **online update check** support based on GitHub releases.
- Fixed **settings UI overflow** in the General section.
- Fixed **settings window resizing behavior** when switching between sections.
- Included stability and compile fixes in filesystem-related logic.
